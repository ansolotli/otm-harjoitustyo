## Komentorivitoiminnot

### Testaus

Testit suoritetaan komennolla:

```
mvn test
```
Testikattavuusraportti luodaan komennolla:

```
mvn jacoco:report
```
Kattavuusraporttia voi tarkastella avaamalla selaimessa tiedoston *target/site/jacoco/index.html*.
